ðŸ”¹ In JavaScript, call(), apply(), and bind() are methods available on functions that let you control what this refers to when the function runs.

1. call() Invokes the function immediately. Lets you pass this and arguments individually.

2. apply() Similar to call(), but arguments are passed as an array. Also invokes the function immediately.

3. bind() Does not invoke the function immediately. Instead, it returns a new function with this permanently bound.
